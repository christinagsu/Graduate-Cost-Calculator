== Description ==

Provides a cost calculator for graduate students.

== Installation ==

1. Upload the files to the `/wp-content/plugins/grad-calculator/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the [grad-cost-calculator] short code in any page or post
